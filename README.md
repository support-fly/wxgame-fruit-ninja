# 🍉 水果忍者 - 微信小游戏

一个完整的微信小游戏实现，包含道具系统、关卡系统和丰富的特效。

## ✨ 功能特性

### 核心玩法
- ✅ 触摸切水果
- ✅ 抛物线运动
- ✅ 刀光轨迹
- ✅ 分数系统
- ✅ 连击加成
- ✅ 炸弹机制
- ✅ 漏掉惩罚

### 道具系统
- ❄️ **冰冻时间** - 减速70%，持续5秒
- ×2 **双倍分数** - 分数×2，持续5秒
- ⚡ **水果狂暴** - 生成速度×2，持续5秒

### 关卡系统
8个难度递增的关卡：
1. Lv.1 新手 (100分)
2. Lv.2 初级 (300分)
3. Lv.3 中级 (600分)
4. Lv.4 高级 (1000分)
5. Lv.5 专家 (1500分)
6. Lv.6 大师 (2500分)
7. Lv.7 传奇 (5000分)
8. Lv.8 至尊 (无限)

### 特效系统
- 💥 切水果爆炸粒子
- 💣 炸弹大爆炸
- ✨ 刀光拖尾
- 🎁 道具收集光效
- 📳 震动反馈

### UI系统
- 🎨 动画开始界面
- 📊 实时分数显示
- 📈 关卡进度条
- 🏁 游戏结束界面
