# 🍉 水果忍者 - 微信小游戏

## 游戏介绍

经典的切水果游戏，触摸屏幕切开飞上来的水果，避开炸弹！

## 游戏特性

- ✅ 流畅的触摸切割效果
- ✅ 多种水果类型
- ✅ 炸弹惩罚机制
- ✅ 连击系统
- ✅ 粒子特效
- ✅ 音效和背景音乐
- ✅ 分数排行榜

## 游戏规则

1. 水果从屏幕底部抛出
2. 触摸划过水果即可切开，获得分数
3. 切中炸弹游戏结束
4. 漏掉3个水果游戏结束
5. 连续切中获得连击加分

## 技术栈

- 微信小游戏 Canvas API
- JavaScript ES6+
- 物理引擎（抛物线运动）
- 对象池优化
